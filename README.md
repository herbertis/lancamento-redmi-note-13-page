# 📱 Redmi Note 13 Series – Landing Page

Landing page promocional desenvolvida para o evento de lançamento da linha Redmi Note 13, com foco em impacto visual, engajamento e captação de leads.

O projeto simula uma página oficial de lançamento, destacando os diferenciais dos dispositivos por meio de vídeos, contagem regressiva e interações dinâmicas.

---

## 🔗 Acesse o projeto:
https://herbertis.github.io/lancaomento-redmi-note-13/

---

## 🎯 Objetivo do Projeto

Criar uma experiência digital imersiva para lançamento de produto, aplicando:

Estratégias de pré-lançamento (countdown)

Destaque visual para features

Captação de leads

Layout responsivo

Integração de mídia (vídeos)

---

## 🔥 Principais Funcionalidades

⏳ Contagem regressiva dinâmica para o evento de lançamento

📱 Apresentação dos modelos:

Redmi Note 13

Redmi Note 13 5G

Redmi Note 13 Pro 5G

🎥 Vídeos em autoplay para evidenciar recursos dos aparelhos

📩 Formulário de cadastro com consentimento de privacidade

📲 Layout totalmente responsivo (desktop e mobile)

🔗 Botões de compartilhamento para redes sociais

---

## 🧠 Conceitos Aplicados

Hierarquia visual voltada para conversão

Destaque estratégico de features

Estrutura modular de seções

Manipulação de DOM para countdown

Uso de vídeo como elemento de storytelling

Responsividade com abordagem flexível

---

## 💻 Tecnologias Utilizadas

✅ HTML5 – Estrutura semântica

🎨 CSS3 – Flexbox, Grid e responsividade

⚙️ JavaScript (Vanilla + jQuery) – Interações e lógica de contagem regressiva

🎥 Vídeos embutidos (.mp4)
