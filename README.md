# 📱 Redmi Note 13 Series - Landing Page

Landing page promocional desenvolvida para o evento de lançamento da linha **Redmi Note 13**, com foco em experiência visual, contagem regressiva, vídeos e interação direta com o usuário.

![Redmi Note 13 Banner](./assets/media/logo.png)

## 🔥 Destaques do Projeto

- Contagem regressiva dinâmica para o evento de lançamento.
- Apresentação dos modelos Redmi Note 13, 13 5G e 13 Pro 5G.
- Integração de vídeos com autoplay para destacar recursos dos aparelhos.
- Formulário para cadastro de e-mail com consentimento de privacidade.
- Layout responsivo para desktop e mobile.
- Botões de compartilhamento para redes sociais.

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (nativo + jQuery)
- 🎥 Vídeos embutidos (formato `.mp4`)
- Responsividade com CSS Flexbox e Grid
